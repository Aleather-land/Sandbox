Rules

  1. Players must register with their full name, preferred name or nickname, and email address.
  2. Each active player may create one event column per week.
  3. Events must occur between 1700 New Zealand Time on Friday and 0800 New Zealand Time on Monday.
  4. All events and predictions must be entered before 1700 New Zealand Time on Friday.
  5. No new entries, edits or predictions are allowed during the event window.
  6. Each challenge must have a clear result so that one winner can be chosen.
  7. After Friday 1700 New Zealand Time, event owners can record the actual result and select the winning prediction.
  8. Each event owner must specify a prediction format so players know how to answer.
  9. Each event owner may add an optional tie-break question. The default is none.
  10. Identical main predictions are only allowed when that event has a tie-breaker. Without a tie-breaker, predictions must be unique.
  11. The winner of each event category gets one point. Everyone else gets zero for that category.
  12. Players may withdraw from the weekly game if they choose.
  13. Previous weeks remain available in the week selector. Event entries and predictions are read-only, but event owners can still select missing winners and add actual results.
  14. The Sports Master weekly tie-break game can be used if the overall weekly points table is tied.
